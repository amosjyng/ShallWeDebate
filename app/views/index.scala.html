A new way to debate

This is currently a work-in-progress. You can check out what's been done so far, though please note that any arguments you make at this time may not be preserved when the site is finished.

The source code to this website will be released on Github once it is finished.

Reuse

Debate once, settle everywhere. Instead of debating the same issue with different people in different places, simply link everyone to the same discussion.

Collaborate

Everyone can contribute. Not convinced by an argument? See some supporting evidence not mentioned? Add it for all to see.

Organize

Cut through the rhetoric to see how arguments really relate to one another, and prevent arguments from falling through the cracks. Unresolved arguments are easy to find and reply to.

Mediate

Outnumbered and out-administrated, you'll never win an argument against X in a forum where everyone believes in X. This site serves as a neutral battleground between all who believe and don't believe in X.
